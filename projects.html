VR Visual Field Test

VR Visual Field Test is a Unity3D application written to emulate the Humphrey Field Analyzer (HFA) II's visual field test.
This test was designed using the FOVE virtual reality headset, because of its integrated eye-tracking.
For more information about visual field testing, please refer to this article written by the Glaucoma Research Foundation (GRF).

Example pattern that the test iterates over. Only one stimulus (white dot) is visible at any one time. The yellow dot in the middle is a focal point for the patient to focus their eye on.

The general flow of the test. This visualizes the process that is followed each time a stimulus is displayed and responded to (or not responded to).

Example of the application's user interface as seen by the test administrator.
'Threshold Chart' shows established thresholds of stimuli.
'Current Stimulus' is a blue circle that surrounds the stimulus that was last displayed to the patient.
'Fixation Graph' is a bar graph that shows the fixation of the patient over time. The bars will directly relate to the distance from the middle focal point to the patient's gaze. These bars are populated dynamically at a rate of 1 per second. 'Current Gaze' shows a real-time representation of where the patient's pupil is looking. This makes it possible for a test administrator to tell if a patient is deviating from the center focal point.

Output generated by the application. This is a full-length test taken by myself with example patient information entered.
(Eye-tracking disabled)

Kickstarter Grabber

A Node.js script that grabs new comments, updates, and Kickstarter projects for a defined profile page. This is run every few minutes using Linux crontab on my remote server.
A webhook is then used to push these new comments, updates, and projects to a Discord server.
Node.js packages including fs, request, and cheerio are utilized.

You can view a public version of this project on BitBucket.

Example of the script's first run with a defined kickstarter profile. This establishes a list of current comments, projects, etc. with which to compare to later.

Example of various alerts coming from the script to Discord after new comments and projects are found.

TTS DeckBuilder

TTS Deck Builder is a small java application to aid in creating deck images for Tabletop Simulator.
Images are copied from the system clipboard, added to a list of stored images and then exported to a large tiled image with black borders. The grid output is 10x7 with the last (70th) position reserved for a card-back that is used for when the cards are in an opponent's hand.
For example purposes the images used will be solid colors.

This is a personal project that is still in development. You can check out this project's development on its GitHub page.

Blocked Off

Blocked Off is a Unity3D top-down puzzle game where you control a green cube. Avoid enemies to reach the goal at the end of the current level to progress to the next. Many more enemy types are planned to be added along with more complex puzzles that will require multiple steps of interaction to complete.

Introduction level which simply states the objective and how to play.
No moving obstacles.
In the next two levels the player must deal with both Red enemies and Green neutral entities. Red enemies follow a pre-determined path, causing the player to start the level over if hit. Green "moving wall" entities move in a pre-determined path and do no inherent harm to the player character.

Purple enemies will follow the player using a simple follow script which also increases its movement-speed over time. The player character has a maximum speed, but the Purple enemies do not. Given time, the Purple enemies will catch up to the player.

Level 5 is the last level in the "tutorial" section of the game. This is also the first level where a player must deal with two different kinds of enemies at once. Currently the game only consists of this tutorial section.

Here we see a simple "You Win" level where the player can move around and knock the yellow particle blocks off the map.

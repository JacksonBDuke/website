Currently Reading

Clean Code: A Handbook of Agile Software Craftsmanship

by Robert C. Martin

While looking for textbooks on C# and Java this book kept coming up as a must-read for any passionate software developer. I am glad I picked it up as it gives very good suggestions on best-practices.

C# 7.0 in a Nutshell: The Definitive Reference

by Joseph Albahari and Ben Albahari

Most of my knowledge of C# comes from my experiences with Java and the use of scripting in the Unity3D engine. This book has helped me understand key differences between the languages, as well as learn about the language in general.

Java: A Beginner's Guide, Seventh Edition

by Herbert Schildt

I picked this up as a refresher to the core concepts and to hopefully learn many new things about Java. The beginning of this book is more focused on someone new to programming, but the book provides ample information on the inner workings of the Java language.
